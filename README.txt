TITLE: QAM2 Task 1 - Database Management Application
PURPOSE: The purpose of this application is to enable users to add, update, and delete customers and appointments in a robust and accessible manner.
AUTHOR: Luke Elrod
CONTACT INFORMATION: luke.elrod.career@gmail.com
VERSION: 1.0.1
DATE: 7/7/2022
IDE: IntelliJ Community 2022.1.2
JDK: Java SE 18.0.1.1
JAVAFX: javafx-base:18
MYSQL CONNECTOR: mysql-connector-java:8.0.29

HOW TO RUN: 
1. Install JDK version 18
2. Unzip the archive
3. Open the folder as a project in intellij
4. Make sure the project default JDK is set to version 18
5. Run the program from the luke.qam2.Main main method.

ADDITIONAL REPORT: The additional report displays each country and the total number of customers from each respective country.
